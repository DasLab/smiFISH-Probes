Probe Filters

2. The GC-content of each probe is calculated based on a percente values. Probes with a GC-content of 40-60% are kept, the others are filtered out.

AACGACAGGCCCTCAGAGTC GC-Content: 65%

ACAGCAAGTACCATGGACCA GC-Content: 50%

3. Thermodynamic values for DNA/RNA hybrids are calculated (Nucleic Acids Research). Median thermodynamic value is used as the top ranked probe, others are ranked based on its distance away from the top ranked probe.

4. Remove lower ranked probes that are overlap on the target RNA with higher ranked probes. The ranked is based off the thermodynamic values explained in the last step.

5. Filter remaining complementary probes through five heuristic probe design rules (PNAS). If these rules are too strict, the rule with the least amount of failed probes (least stringent rule) is removed. This is repeated until there is the desired amount of probes or more.

Final Probes

6. Probes that pass these filters are called "final probes". The final probe set is able to be downloaded as a CSV file to be sent to a DNA Oligo ordering site.

Five Heuristic Probe Desgin Rules (PNAS)

1. Filter out nucleotide compositions with A < 28%
2. Remove anything with AAAA stacks
3. Enforce C composition between 22-28%
4. No CCCC stacks in any 6 consecutive nucleotides in the first 12 positions
