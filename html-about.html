❓ About RNA Probe-ono

RNA Probe-ono is a locally-run website that utilizes Pyodide to run a Python script that automatically generates probes for single-molecule inexpensive RNA fluorescent in situ hybridization

(smiFISH)

experiments. The Python script, developed by the Das Lab at Stanford University, takes an inputted RNA sequence and outputs an excel file of probes ready to be uploaded to online DNA ordering systems. The Python software and locally-run website help remove the problematic task of probe generation and allow the general science community to conduct single-molecule RNA detection experiments.
